说明：

1、查看帮助文档请使用浏览器打开/doc/index.html

2、demo中仅仅演示了函数的功能和使用方法，需要在实际开发过程中完善。

3、具体开发时，请先将libs目录下thm-fingerprint-sdk-release-5.0.jar和so库文件拷贝至工程libs目录下的对应文件夹。

4、如果demo源文件出现中文乱码，请设置：Project ->Properties,Resource中，"Text file coding"设为  “UTF-8”，OK。

5、如果doc/的帮助文档中文乱码，请在浏览器打开后，设置页面编码为：Unicode。

6、libs/thm-fingerprint-sdk-release-5.0.jar为编译的release版sdk



English Version:

1、please use the browser to open the file of“/doc/index.pdf”.

2、The demo just demonstrates the function and instructions of functions, need to improve it in the actual developing process.

3、When developing actually, first copy the “thm-fingerprint-sdk-release-5.0.jar” in “libs” and “so” library files to the folder corresponding to project “libs” directory.

4、If the “demo” source files appears Chinese garbled, please set: Project -> Properties, in Resource, "Text file coding" is set as "UTF-8", OK.

5、If the “doc/”  help files are Chinese garbled, after you open the browser, and set encoding of the page as: Unicode.

6、libs/thm-fingerprint-sdk-release-5.0.jar is the compiled release version sdk.
